Gaijin ======

Remove the pesky need to visit all those sites and have 1 bot do it all for you!
ping - Check if the bot is online, display the bot's ping in ms, and view the amount of servers the bot's in
shards - View all shard latencies
help - Display this menu
updates - View the current or a past version's changes
syntax - View the valid syntax for date paramaters
caps - Manipulate the capitalization of text
space - Space out text
vowels - Remove all consonants from text
consonants - Remove all vowels from text
corrupt - Use the Zalgo method to corrupt characters
decorrupt - Remove the Zalgo characters from a string
flip - Flip a string
unflip - Unflip a string from any direction (Will not unreverse)
leet - Change leet-able characters to leet speak in text
deleet - Turn leet speak into letters in text
morse - Convert text to Morse Code
unmorse - Convert More Code to text
binary - Convert text to Binary
unbinary - Convert Binary to text
encode - Encode text into Base64
decode - Decode Base64 into utf8 text
circle - Put all characters in the provided text in circles
uncircle - Remove circles from the provided text
aesthetic - Give the text some ａｅｓｔｈｅｔｉｃ
unaesthetic - Remove the ａｅｓｔｈｅｔｉｃ from text
double - Change the font to a doubled font in the text
undouble - Undouble doubled characters in text
emojify - Turn numbers and letters in the text into emojis
say - Make the bot say whatever you want
ascii - Create ASCII text
replacespace - Replace all the spaces in the provided text with something
math - Solve a math equation | [Symbol Key](https://github.com/silentmatt/expr-eval)
rand - Pick a random integer between 2 numbers
replace - Find something in text and replace it with something else
scramble - Scramble the letters of text
indices - Return a list of all the indices of occurances a phrase has in the given text
amount - Return the amount of occurances a phrase has in the text provided
length - Return the length of the given text
differences - Find the differences between 2 given texts
choose - Choose 1 item from a list of items provided
resources - List links to some useful websites
repeat - Repeat provided text (one message)
tempconvert - Convert from Fahrenheit to Celsius and vice versa
noformat - Return the last message sent by the bot in a codeblock for easy copying
dblwidget - View a customized widget of the bot (Discord Bot List)
